O foco deste trabalho é desenvolver um aplicativo móvel para sistema operacional Android, que colete dados de sinal vocal e informações do paciente, utilizando os recursos de um aparelho celular e sua rede. O aplicativo se comunica com um servidor atrav\'es de uma nuvem, assim todo o processamento é realizado no servidor.
O projeto é realizado em 4 etapas:

1) Escolha do sistema operacional e do servidor;

2) Desenvolvimento da aplicação (servidor, servidor web e aplicativo);

3) Aquisição e processamento dos dados utilizando correlação;

4) Apresentação dos dados na interface gráfica do servidor, no site web e no aplicativo para o usuário. Ao final foi possível estimar a frequência fundamental da voz utilizando o método de correlação e também atualizar as interfaces gráficas do servidor, servidor web e aplicativo.
